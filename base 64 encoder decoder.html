HOW THIS WORKS:

1. Convert Data to Binary - Each character in the input text is converted to its 8-bit binary form.
2. Group into 6-bit Chunks - The binary data is split into chunks of 6 bits (instead of the usual 8-bit bytes).
3. Map to Base64 Characters - Each 6-bit chunk is converted into a corresponding Base64 character using this set: ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789+/
4. Padding if Necessary - If the input isn't a multiple of 3 bytes, "=" is added at the end to make it a multiple of 4 Base64 characters.
Example: Encoding "Hi" to Base64:
1. ASCII to Binary: H → 01001000, i → 01101001 Binary: 0100100001101001
2. Group into 6-bit chunks: 010010 000110 1001
3. Convert to Base64 Characters: 010010 → S, 000110 → G, 100100 → k
4. Add padding (if needed): "Hi" (2 bytes) is not a multiple of 3, so it needs one "=": Base64 Output: "SGk="
